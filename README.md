# 🚀 Self-Healing System (Auto Recovery Tool)
Self-healing system built with Python that monitors processes and automatically restarts failed services to ensure system reliability and uptime.
### 👨‍💻 Developed by PREMSANKAR PRADHAN

---

## 📌 Overview

This project is a Python-based self-healing system that continuously monitors running processes and automatically restarts them if they fail. It is designed to improve system reliability, reduce downtime, and minimize manual intervention.

---

## ⚙️ Features

* 🔍 Real-time process monitoring
* 🔄 Automatic restart of failed services
* 📝 Logging system for tracking events
* ⚡ Configurable monitoring interval

---

## 🛠️ Technologies Used

* Python
* psutil
* OS-level process management

---

## 🚀 How It Works

1. The system continuously checks if a specific process is running
2. If the process stops, it automatically restarts it
3. All events are logged for monitoring and debugging

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```
